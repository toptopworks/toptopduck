# Conversation history: turn-windowed (recent N full, far summarized), unified with result_N schema window

## Decision

每轮 prompt 的**上下文窗口按「轮次」统一裁剪**，不再把"对话历史"与"`result_N` schema"当两个独立窗口：

1. **最近 N=20 轮发全量**：该轮用户提问 + LLM 响应（SQL + `assumption` 文本）+ 该轮产出 `result_N` 的全 schema（含样本）。全部源 Dataset 全 schema 始终全量发送（不裁剪）。
2. **窗口外轮次发摘要**：一行 = 该轮提问摘要 + 产出 `result_N` 名（**不发 SQL、不发全 schema**）；文本态轮次（ADR-0018 澄清 / ADR-0017 拒绝）摘要为"澄清/拒绝了 X"。
3. 窗口外 `result_N` 仍可被自然语言点名重定向（ADR-0010）——靠的就是这一行提问摘要 + 表名/列名。

## Context

ADR-0013 给 `result_N` schema 做了窗口化，但**对话历史文本**（提问 + LLM 响应）是另一个无界增长源——ADR-0009/0010 把它列为 LLM 输入却无窗口/摘要策略。长会话下历史 + K=20 schema 撞 token 上限 / 成本失控。且一轮产一个 `result_N`，ADR-0013 的"`result_N` 远期摘要 = 提问摘要"与"历史远期轮次摘要"本就是同一句话——双轨冗余。

## Why

1. **统一单轨消除冗余（DRY + KISS）**：按轮次裁剪，每轮的全量信息（提问 + 响应 + 该轮 `result_N` schema）打包，窗口外只留摘要——不再维护"历史文本窗"和"schema 窗"两个重叠列表。
2. **远期摘要保住远期指代**：提问摘要让 LLM 把"上上周那个去重表"映射到 `result_N`，与 ADR-0010/0013 的自然语言重定向同源。
3. **窗口外不发 SQL**：SQL 语义可由 `result_N` schema 重建，省 token，且避免远期失效 SQL 误导。
4. **源 Dataset 全 schema 不裁剪**：源是分析锚点、数量少（用户主动上传），全量发送成本可控、收益（SQL 正确性）高。

## Considered options

- **全量历史，不裁剪**：上下文最完整，但长会话撞 token 上限 / 成本失控、远期无关历史干扰 SQL 生成。**否决**。
- **保留 0013 双轨 + 独立历史文本窗**：不校准 0013，但留两个重叠窗口的冗余。**否决**。
- **纯滑动窗口、远期无摘要**：丢远期指代能力，"上上周那个表"重定向断。**否决**。
- **窗口外发完整 SQL**：冗余（schema 可重建语义）且失效 SQL 误导。**否决**。

## Consequences

- **校准 ADR-0013**：窗口计数口径从"按 `result_N`（K=20）"统一为"按轮次（N=20），每轮含该轮 `result_N` 全 schema"——语义不变，ADR-0013 Consequences 已追加校准指针。M=100（同时存在 `result_N` 上限）不受影响。
- N=20 为默认，可调；与 ADR-0013 的窗口同为"近窗全量 + 远窗摘要"哲学。
- 提问摘要（喂 LLM 与给用户看同源，ADR-0010）是远期轮次窗口外信息的全部载体——其质量直接决定远期重定向能力，须保证摘要忠实。
- 长会话下 prompt 成本有界（源全量 + 最近 20 轮全量 + 远期单行摘要）。
- **被 ADR-0028 延伸**：N=20 窗口从「仅结果轮 + 文本态轮」扩展为「全部四类 outcome（结果/文本态/失败/取消）均占槽」，窗口外载荷按 outcome 类型给出（失败→「失败：{原因}」、取消→「已取消」）——本 ADR 已为文本态轮给的一行摘要模式直接复用。
- **被 ADR-0039 校准**：「提问摘要」= **用户原话的有界截断（首句 / 截断），非 LLM 再生成**；窗口内仍发全量原话（不变），仅窗口外用截断——天然忠实（= 用户原话）直接满足本 ADR「须忠实」，零额外 LLM 调用。
- **被 ADR-0040 澄清**：N=20 窗口「按轮次」明确**排除源生命周期事件**（加/换/删源，非轮次）——对话历史全量含之、LLM 窗口从中只取轮次；源事件携带的工作集变更已由「源全量始终发送 + stale 派生不进工作集」覆盖，不占窗无损语义。见 ADR-0040。
